Ejemplos de Propiedades de Texto en CSS

1. text-align-last

Este párrafo tiene su texto alineado a la izquierda, pero el último renglón está alineado a la derecha.

2. text-decoration

Este texto está subrayado, y el subrayado es de color rojo y de estilo ondulado.

3. text-emphasis

Este texto tiene énfasis en forma de punto azul debajo.

4. text-indent

Este texto está indentado en su primer renglón.

5. text-justify

Este texto está justificado, y la justificación se aplica entre palabras.

6. text-shadow

Este texto tiene una sombra que lo hace destacar.

7. text-overflow

Este es un ejemplo de texto que se desborda. Si el texto es demasiado largo, se mostrará con puntos suspensivos al final.

8. text-wrap

Este texto tiene un envolvimiento que lo equilibra dentro de su contenedor.
