Color

Establece el color de los elementos (textos, fondos).

  • Por nombre: rojo en css, se puede espesificar un color utilizando un nombre de color predefinido.
  • HEX: #ff6347
    • en css, se puede espesificar un color utilizando un valor hexadecimal.
  • RGB: rgb(255, 0, 0)
    • en css, un color se puede especificar como un valor rgb.
  • HSL: hsl(10, 100%, 50%)
    • en css, se puede espesificar un color mediante tono saturacion, y luminosidad(hsl).

Opacidad

Determina cuán transparente es un elemento.

  • transparent image
    • la propiedad de opacidad puede tomar un valor de 0.0-1.0. cuanto menor sea el valor, mas transparente sera.
  • transparent hover effect
    • la propieadad de opacidad a menudo se usa junto con el selector :hover para cambiar la opacidad al pesar el mouse.
  • transparent box
    • al utilizar la propiedad de opacidad para agregar transparencia al fundo de un elemento, todos sus elementos secundarios heradan la misma transparencia.
  • transparency using rgba
    • si no desea aplicar la opacidad a los elementos secundarios como en nuestro ejemplo anterior, utilice valores de color rgba.
