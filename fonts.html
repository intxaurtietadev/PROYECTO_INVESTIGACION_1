Como aplicar diferentes tipografias

Servicio de distribución de fuentes: Google Fonts: Una vez que hayamos explorado la biblioteca de fuentes y hecho nuestra selección, necesitaremos copiar el código proporcionado por el servicio de distribución en la etiqueta de nuestro HTML. Google Fonts proporciona algo como esto: Si observamos el archivo CSS (en la URL mencionada arriba) que nos ha proporcionado Google Fonts, Adobe Fonts u otro proveedor, notaremos que incluye una declaración @font-face para cargar el archivo de la fuente y asignarlo a una familia, un estilo (normal o itálico) y un peso. En su forma más básica, el CSS se verá algo así, y es prácticamente el mismo CSS que habríamos escrito nosotros mismos si no estuviéramos utilizando una red de distribución de fuentes. @font-face { font-family: 'FAMILY_NAME'; font-style: NORMAL_OR_ITALIC; font-weight: NUMERIC_WEIGHT_VALUE; src: url(FONT_FILE_NAME.woff2) format('woff2'); }

Referenciar los archivos de fuente

En nuestro archivo CSS, primero necesitaremos crear una referencia de familia para los archivos de fuente que estamos cargando mediante una declaración @font-face. (Recomendamos usar un archivo CSS dedicado exclusivamente para cargar nuestras fuentes). Aunque la ruta al archivo de fuente debe ser precisa, podemos nombrar nuestra familia como queramos. No es necesario que esté relacionada con el nombre real de la tipografía, aunque, por supuesto, es recomendable mantenerlos consistentes. (Ten en cuenta que no debe haber un espacio entre url y el paréntesis de apertura; este es un error común). @font-face { font-family: 'FAMILY_NAME'; src: url(FONT_FILE_NAME.woff2) format('woff2') }

Asignar pesos y estilos

A continuación, agreguemos un estilo, ya sea “normal” (es decir, recto), “italic” o “oblique”, y declaremos un peso numérico para la fuente: @font-face { font-family: 'FAMILY_NAME'; font-style: NORMAL_OR_ITALIC; font-weight: NUMERIC_WEIGHT_VALUE; src: url(FONT_FILE_NAME.woff2) format('woff2'); } Deberemos repetir estos pasos para cada archivo de fuente que deseemos cargar, pero ten en cuenta que el nombre de la familia permanece igual, a menos que realmente estemos cambiando de tipografía. Además, ten en cuenta que las declaraciones de peso y estilo se manejan de manera ligeramente diferente para fuentes variables.
