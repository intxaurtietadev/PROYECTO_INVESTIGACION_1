Propiedades de las fuentes

Las propiedades CSS de las fuentes son las que permiten controlar el tamaño, el tipo, el grosor o el estilo de las letras, entre otras cosas. Aunque hay numerosas propiedades, las siguientes podrían ser las más destacadas.
